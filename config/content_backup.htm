stage1

實習目的:

  1. 讓學員利用電腦輔助機械設計套件執行協同任務, 找出保存各設計資料版本的有效方法
  2. 讓學員利用最有效率的方式展示協同產品設計過程的各相關細節與關鍵技術
  3. 讓學員實習如何協同編寫淺顯易懂且內容充實的產品設計簡報與結案報告 (Reveal.js 與 latex-paper 研究)
  4. 讓學員在協同設計過程, 利用 ssh 維護個人與分組倉儲與網站
  5. 讓學員利用 git remote add 建立 Github 與 Gitlab 上的同步倉儲與協同設計網站
